CSS3-Only Spinner Tests

This is the default spinner created without any arguments to the mixin. It is defined in css-spinner.css. If you do not want to use Sass/Compass, you can just use the CSS file directly.
@include spinner()
These are tests using different sizes. css-spinner-test.css contains these definitions. Each was created using the arguements to the mixin as labeled below.
@include spinner(15)
@include spinner(35)
@include spinner(55)
